## 📞 快速参考

### 显存占用
- 原版：24GB
- 优化版：8GB
- 改进：↓ 67%

### 推理速度
- 原版：2.5s
- 优化版：0.8s
- 改进：↓ 68%

### Token 数量
- 原版：120k
- 优化版：20k
- 改进：↓ 83%

### 关键 API
- `reset_cache()` - 重置缓存
- `prepare_canonical(points_3d)` - 前置准备
- `_build_rotation_from_normal(normal)` - 旋转矩阵

### 关键参数
- `topk_views=4` - 视角筛选
- `feat_agg_dim=256` - 特征维度
- `feat_agg_layers=1` - Transformer 层数
